# vscode-ngseke-snippets

## Disclaimer

This is a slightly modified fork of [vscode-standardjs-snippets](https://github.com/capaj/vscode-standardjs-snippets) by [Jiri Spac](https://github.com/capaj).

## How to debug

Press F5

## How to publish

```
npm install -g @vscode/vsce
vsce login ngseke
vsce package
vsce publish
```
